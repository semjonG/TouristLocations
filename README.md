## Выполнено

 Приложение запрашивает доступ к геопозиции пользователя и передает её в другое приложение (2GIS) для построения маршрута к выбранным локациям, которые были получены с сервера.
 
- MVVM
- SwiftUI
- Combine 
- CoreLocation
- DeepLinking
